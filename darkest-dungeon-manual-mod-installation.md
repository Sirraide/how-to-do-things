Use 'View Game Files' or whatever it's called on steam to find the game directory; there should be a 'mods' directory in there; the mods go in there. Each mod must be a single directory within 'mods', with all the mod's files placed within its directory. When downloading a zipped mod, make sure to create a subfolder for it if it doesn't already have one.

On linux this doesn't work for some reason if you're using the native build, so switch to the most recent Proton version.
